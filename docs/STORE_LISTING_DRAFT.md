# 插件市场上架信息草案

## 插件名称

病历助手

## 短描述

本地运行的感染科语音病历草稿输入、模板和核对助手。

## 详细描述

病历助手是一款面向感染科临床场景的本地语音输入工具。医生可以在独立扩展页面中进行语音听写、结构化模板填写、多患者草稿管理、热词维护和风险核对，然后将确认后的文本复制到医院病历系统。

核心特点：

- 本地语音识别：默认连接同一台电脑上的本地识别服务；
- 医生核对优先：所有内容以草稿形式呈现，提交前必须由医生确认；
- 感染科热词：支持病原体、抗感染药物、检查指标等热词维护；
- 模板管理：支持主诉、现病史、流行病学史、辅助检查、诊疗经过等模板；
- 多患者草稿：便于临床场景中临时切换和核对；
- 停顿标点：根据短暂停顿/长停顿自动补充逗号和句号；
- 隐私友好：默认不上传语音和病历内容到开发者服务器。

## 医疗免责声明

本工具仅用于辅助录入和整理病历草稿，不提供诊断、治疗建议或自动病历生成结论。识别结果可能存在错误，必须由医生最终核对。

## 权限说明

- `storage`：保存用户本地模板、草稿、版本历史和设置；
- `host_permissions: http://127.0.0.1:8765/*`：连接用户本机语音识别服务；
- 麦克风访问：由浏览器页面在用户点击“开始听写”后请求，用于采集语音并发送到本地服务。

## 免费版 / Pro 版说明

免费版包含基础语音输入、草稿编辑、复制、TXT 导出、基础模板和本地热词。后续 Pro 版可包含批量模板、团队热词包、高级质控规则、机构部署管理和离线授权等能力。涉及病历安全、隐私保护和基本核对的能力不应被完全放入付费墙。

## 审核说明

本扩展依赖本地 Windows 服务。审核时请先启动本地服务，再打开扩展页面。服务健康检查地址：`http://127.0.0.1:8765/health`。

## 多语言说明

插件已支持中文/英文界面切换，并提供中文普通话与英文听写选择。中文听写为当前主要能力，支持流式输入；英文听写为 Beta 能力，依赖本地英文 ASR 模型，当前采用停止录音后批量识别。插件 manifest 已使用 `_locales` 提供中文与英文商店基础信息。
