# GitHub Pages 隐私政策 URL 准备说明

当前已经准备好可直接托管的静态页面：

- `docs/index.html`
- `docs/privacy-policy.html`

建议 GitHub 仓库名：`bingli-assistant`

如果使用 GitHub Pages，并选择从 `main` 分支的 `/docs` 目录发布，则隐私政策 URL 通常是：

```text
https://atlantis379.github.io/bingli-assistant/privacy-policy.html
```

如果仓库名不是 `bingli-assistant`，请把 URL 中的仓库名替换为实际仓库名。

## 上架前必须替换

隐私政策页面中仍有一处待替换内容：

```text
上架前替换为你的支持邮箱
```

请在 Edge 插件市场提交前替换为真实支持邮箱或支持页面。

## GitHub Pages 开启步骤

1. 在 GitHub 创建仓库，例如 `bingli-assistant`。
2. 将本地仓库推送到 GitHub。
3. 进入仓库 `Settings` → `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/docs`。
6. 保存后等待 GitHub Pages 构建完成。
7. 打开生成的隐私政策 URL，并复制到 Microsoft Partner Center。
