# 发布检查清单

## A. 代码与权限

- [ ] `manifest.json` 使用 Manifest V3；
- [ ] 只申请 `storage` 和 `127.0.0.1:8765` 等必要权限；
- [ ] 不加载远程 JavaScript；
- [ ] 插件 ZIP 根目录直接包含 `manifest.json`；
- [ ] 浏览器控制台无明显错误；
- [ ] 服务未启动时，插件有清晰引导；
- [ ] 服务启动后 `/health` 正常返回；
- [ ] `/license/status` 正常返回免费版状态；
- [ ] 流式识别、批量降级、停顿标点均可用。

## B. 隐私与合规

- [ ] 准备公开可访问的隐私政策 URL；
- [ ] 商店数据使用声明与隐私政策一致；
- [ ] 明确说明语音和病历内容默认不上传开发者服务器；
- [ ] 明确本地保存位置和删除方式；
- [ ] 明确医疗免责声明；
- [ ] 完成第三方许可证清单；
- [ ] 核对 FunASR、模型、依赖包许可证。

## C. 分发

- [ ] 生成浏览器扩展 ZIP；
- [ ] 生成 Windows 本地服务安装包或压缩包；
- [ ] 准备模型首次下载/离线包说明；
- [ ] 准备卸载说明；
- [ ] 准备版本号、更新日志和回滚方式；
- [ ] 准备内测用户安装文档。

## D. 商店素材

- [ ] 图标 16/48/128 px；
- [ ] 至少 3 张截图：编辑页、语音输入、模板/热词/多患者；
- [ ] 短描述、详细描述、权限说明；
- [ ] 隐私政策链接；
- [ ] 支持邮箱/官网链接；
- [ ] 如果有收费功能，准备测试授权或审核说明。

## 多语言说明

插件已支持中文/英文界面切换，并提供中文普通话与英文听写选择。中文听写为当前主要能力，支持流式输入；英文听写为 Beta 能力，依赖本地英文 ASR 模型，当前采用停止录音后批量识别。插件 manifest 已使用 `_locales` 提供中文与英文商店基础信息。
