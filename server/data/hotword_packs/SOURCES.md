# 医用专业词汇包来源说明

更新日期：2026-06-24

## 选词原则

- 只收录病历输入和感染科场景中高频、相对稳定的医学术语；
- 内置词库仅用于语音识别热词提示，不自动生成诊断或治疗建议；
- 药物采用通用名，不采用商品名；
- 对于可能随指南变化的治疗策略，不作为词库判断依据；
- 用户自定义词库与内置词库分离，便于医生或医院自行维护。

## 主要参考来源

1. 国家卫生健康委及原国家卫生计生委发布的抗菌药物临床应用相关管理文件与《抗菌药物临床应用指导原则（2015年版）》：用于抗菌药类别、常用通用名和感染治疗常见表述的筛选。
2. WHO AWaRe / Essential Medicines 相关抗菌药分类资料：用于核对常见抗菌药英文/通用名范围。
3. CDC antimicrobial resistance / healthcare-associated infection 公开资料：用于核对常见耐药菌、医院感染相关病原体范围。
4. 感染科、呼吸道感染、血流感染、泌尿系感染、真菌感染等临床病历常用术语：用于通用医学和感染科词库筛选。

## 维护说明

- `general_medical.txt`：通用医学词库；
- `infectious_disease.txt`：感染科场景词库；
- `antimicrobials.txt`：抗菌药/抗感染药通用名；
- `pathogens.txt`：常见病原体；
- `user_custom.txt`：用户自定义词库，可由插件导入/导出。

发布前如用于商业分发，建议由感染科医生或药师进行一次院内复核。
